
## Tech Stack:

- **Backend**: Node.js + Express, PostgreSQL, JWT, Multer
- **Frontend**: React + Vite, Redux Toolkit, TailwindCSS, Framer Motion
## Setup

### Prerequisites
- Node.js 
- PostgreSQL (or Cloud Db )

### Backend
```bash
cd backend
npm install
cp .env.example .env
npm run migrate   
npm run dev
```

### Frontend
```bash
cd frontend
cp .env.example .env
npm install
npm run dev
```
